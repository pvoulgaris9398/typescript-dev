# TypeScript Notes

- `npm init -y`
  - Adds `package.json`
- `npm install typescript --save-dev`
  - Adds `typescript` to node_modules and `package.json`
- `npm tsc --init`
  - Adds `tsconfig.json`
- Install `tsx` and run `npx tsx src/index.ts`
- See [tsx](https://tsx.is/getting-started)
